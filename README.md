# All4Sec

Institutional website for All4Sec developed with Angular e Tailwind CSS.

## Stack

- TypeScript
- Angular 21
- Tailwind CSS
- EmailJS

## Requirements

- Node.js (LTS)
- npm

## Locally

```bash
npm install
```

Development server:

```bash
npm start
```

```text
http://localhost:4200
```

## Build

```bash
npm run build
```
